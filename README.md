# doron-calculator
מחשבון ערך עתידי - נתוני אמת
סימולטור מקצועי המחשב תשואות צפויות על בסיס ביצועי שוק היסטוריים בין השנים 1995-2024.

פותח על ידי: דורון שרייבמן
גילוי נאות: המחשבון נועד למטרות הדגמה וחינוך פיננסי בלבד. התוצאות המוצגות מתבססות על נתוני תשואה היסטוריים בלבד (1995–2024) ואינן מהוות תחזית, הבטחה או התחייבות לתשואה עתידית כלשהי. ביצועי עבר אינם ערובה לביצועים עתידיים. החישובים אינם מביאים בחשבון את מצבו הפיננסי האישי של המשתמש, מטרות ההשקעה, אופק ההשקעה, רמת הסיכון המתאימה לו, חבות המס הרלוונטית, עלויות עסקה, או כל גורם אישי אחר. אין לראות בתוצאות המחשבון משום ייעוץ השקעות, ייעוץ פנסיוני, ייעוץ מס או כל ייעוץ פיננסי אחר. השימוש במידע המוצג הינו באחריות המשתמש בלבד. לפני כל החלטת השקעה מומלץ להתייעץ עם יועץ השקעות מורשה בעל רישיון כדין. ט.ל.ח.
